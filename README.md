# workflows

Just some simple workflow examples.

# installation

`make all`

# snakemake workflow

`cd snakemake/ && ./run.sh`

# nextflow workflow

`cd nextflow/ && ./run.sh`

# CWL workflow

`cd cwl/ && ./run.sh`
